# Agentic Quality Control & Perfection System

## Overview

The agentic system ensures **perfect prompt adherence** and **guaranteed quality** for every image generated by SDX. It leverages the **penta text encoder system** (T5-XXL + 4 CLIP variants) for semantic validation across 5 independent perspectives.

---

## Architecture

### The Penta Text Encoder System

SDX uses 5 text encoders, each capturing different semantic aspects:

| Encoder | Hidden Dim | Purpose | Semantic Focus |
|---------|-----------|---------|-----------------|
| **T5-XXL** | 4096 | Sequence understanding | General semantics, long-form meaning |
| **CLIP-L** | 768 | Basic vision-language | Simple visual concepts |
| **CLIP-bigG** | 4096 | Fine-grained details | Visual detail richness |
| **CLIP-H** | 1024 | High-level scenes | Composition & layout |
| **LongCLIP-L** | 768 | Extended context | Full prompt context |

## Quality Metrics

### Overall Score Calculation

```
Overall Score = (
    Adherence × 0.4          (Prompt matching - critical)
    + Consistency × 0.3      (Encoder agreement)
    + (1 - Divergence) × 0.15 (Low encoder disagreement)
    + VisualQuality × 0.15   (Aesthetic quality)
)
```

### Quality Tiers

| Score | Status | Action |
|-------|--------|--------|
| **95-100%** | Perfect ✓ | Done, use image |
| **85-94%** | Good | Minor refinements optional |
| **75-84%** | Acceptable | Recommend refinements |
| **<75%** | Poor | Regenerate |

---

## Refinement Actions

When adherence < 85%, the system recommends:

1. **increase_detail** - Add more fine details
2. **improve_lighting** - Better light/shadow balance
3. **enhance_colors** - More vivid color palette
4. **sharpen_focus** - Increase sharpness
5. **improve_composition** - Better layout
6. **increase_contrast** - Better tonal range
7. **refine_textures** - Better surface details
8. **enhance_realism** - More photorealistic
9. **diversify_elements** - More varied elements
10. **add_depth** - Better sense of depth

## Integration with SDX Pipeline

### In sample.py / train.py

from innovations.pipeline import SDXAdvancedPipeline

    prompt,
    t5_embeddings,
    clip_embeddings,

    refined_latent, assessments = quality_system.iterative_perfection(
        prompt, t5_embeddings, clip_embeddings, image_latent

return vae_decode(image_latent)
