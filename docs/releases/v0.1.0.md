# SDX v0.1.0

**Date:** 2026-03-13  
**Tag:** `v0.1.0`  
**PyPI / install:** This repo uses `pip install -r requirements.txt` from source; `pyproject.toml` version aligns with the tag.

## Highlights

- **Diffusion Transformer** training and sampling (`train.py`, `sample.py`), T5 (+ optional triple CLIP), VP diffusion, EMA, validation, many quality flags.
- **2026 industry / architecture docs:** [`LANDSCAPE_2026.md`](../LANDSCAPE_2026.md), [`ARCHITECTURE_SHIFT_2026.md`](../ARCHITECTURE_SHIFT_2026.md), [`WORKFLOW_INTEGRATION_2026.md`](../WORKFLOW_INTEGRATION_2026.md) — plus [`utils/architecture/architecture_map.py`](../../utils/architecture/architecture_map.py) (theme → SDX parity).
- **Training:** optional **`--resolution-buckets`** (multi-res / aspect, single-GPU) with [`ResolutionBucketBatchSampler`](../../data/bucket_batch_sampler.py).
- **Inference / tooling:** `--pick-best` including **`combo_exposure`**, [`orchestrate_pipeline.py`](../../scripts/tools/ops/orchestrate_pipeline.py), [`rag_prompt.py`](../../utils/prompt/rag_prompt.py), [`character_lock.py`](../../utils/consistency/character_lock.py), post-process helpers in [`quality.py`](../../utils/quality/quality.py).
- **Native bridge:** Rust / Zig / C++ helpers under [`native/`](../../native/README.md) with Python shims in [`native/python/sdx_native/`](../../native/python/sdx_native/).
- **Tests:** `pytest tests/` (unit, integration, diffusion).

## Install

```bash
git clone https://github.com/Llunarstack/sdx.git
cd sdx
git checkout v0.1.0
pip install -r requirements.txt
python scripts/tools/dev/quick_test.py
```
