# SDX v0.1.0 — Baseline Text-to-Image Training Framework

**Release Date:** March 13, 2026  
**What it is:** The foundation of SDX—a complete framework for training and sampling text-to-image diffusion transformers on your own data.

---

## What You Get

A Python framework for building custom image generation models:

- **Train** a model on your images + captions using `train.py`
- **Sample** images from your trained model using `sample.py`
- **Support for T5 text encoding** (with optional CLIP models)
- **Variance-Preserving (VP) diffusion** for stable, predictable training
- **Multi-resolution training** with smart batching (different image sizes together)
- **Quality ranking** to pick the best generated images

---

## Key Components

### Training (train.py)
- Load image + caption pairs from disk
- Train a Diffusion Transformer (DiT) model
- Save checkpoints automatically
- Support for single GPU and multi-GPU training

### Sampling (sample.py)
- Generate images from text prompts using your trained model
- Control generation strength with CFG (Classifier-Free Guidance)
- Multiple quality-ranking modes to pick the best results
- Fast inference with various solver options

### Native Acceleration (Optional)
- C++ and Rust helpers for faster operations
- Completely optional—everything works in pure Python
- Includes timestep scheduling, file utilities, and more

---

## How to Use

```bash
# Install
pip install -r requirements.txt

# Train on your images
python train.py --data-path my_images/ --results-dir outputs/

# Generate images
python sample.py --ckpt outputs/best.pt --prompt "a cat" --out cat.png
```

---

## Architecture

```
Your Images → train.py → Model Checkpoint → sample.py → Generated Images
```

Simple linear pipeline: you provide data, we train a model, you sample from it.
