# SDX v4

Release date: 2026-04-06

v4 focuses on test-time intelligence in the TCIS hybrid generation stack and expands advanced-model wiring for stronger control and refinement.

## Highlights

- **Uncertainty-scaled TCIS runtime**
  - Adds uncertainty scoring from consensus confidence, score gap, head disagreement, and constraint quality.
  - Dynamically extends iteration budget when uncertainty is high.
- **Cross-iteration elite memory**
  - Stores recent elite image signatures and adds a diversity bonus in candidate selection.
  - Helps reduce iterative mode collapse in repeated self-correction runs.
- **Constraint annealing**
  - OCR/count/saturation consensus weights can ramp up or down across iterations.
  - Improves early exploration and late-stage strictness.
- **Advanced model stack completion**
  - Added and wired: LongCLIP-L, moondream2, Marigold depth/normals, TAESD/TAESDXL, CodeFormer, Consistency Decoder, ConvNeXtV2-Large, LAION aesthetic predictor, AnyDoor reference.
  - Added downloader hardening and fallback handling for non-standard sources.

## Usage snippet

```bash
python -m scripts.tools hybrid_dit_vit_generate \
  --ckpt results/.../best.pt \
  --vit-ckpt ViT/runs/best.pt \
  --prompt "your prompt" \
  --out outputs/v4.png \
  --iterations 4 \
  --auto-shape-scaffold \
  --pareto-elite \
  --constraint-anneal up \
  --uncertainty-threshold 0.42 \
  --uncertainty-extra-iterations 1 \
  --elite-memory-size 8 \
  --diversity-bonus-weight 0.05
```
