## v7.0.0

Release highlights since **[v6.0.0](v6.md)**: **CI hardening** (basedpyright spot-check, markdown link verification), **tracked `.editorconfig`**, **evaluation recipe** docs, and a small **training artifact** API (`utils.generation.run_artifacts`) so manifest filenames are not duplicated as string literals.

### Engineering

- **`.github/workflows/ci.yml`**: after Ruff, runs `basedpyright --level error` on curated paths, then `python -m scripts.tools verify_doc_links`, then pytest.
- **`.editorconfig`**: editor defaults (UTF-8, LF, trim whitespace, Python/YAML/Markdown indent). Removed from `.gitignore` so `docs/FILES.md` link resolves and clones stay consistent.
- **`utils/generation/run_artifacts.py`**: `RUN_MANIFEST_FILENAME`, `TRAIN_CONFIG_SNAPSHOT_FILENAME`, `train_artifact_paths()`; `train.py` uses these for `run_manifest.json` / `config.train.json`.

### Docs

- **`docs/recipes/quick_eval_holy_grail.md`**: copy-paste flow for one-command demo, optional `sample.py` with Holy Grail preset, and where training writes manifests.
