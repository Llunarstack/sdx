# Civitai model bank (Illustrious / NoobAI, NSFW search)

This folder holds **metadata exported from the [Civitai public REST API](https://developer.civitai.com/docs/api/public-rest)** — not HTML scraping of the search page (results are loaded in the browser).

## Files

- **`nsfw_illustrious_noobai_models.csv`** — One row per model that has at least one `modelVersions[].baseModel` in **`Illustrious`** or **`NoobAI`**, taken from API search `query=nsfw` + `nsfw=true`. Columns: `id`, `name`, `type`, `bases`, `triggers` (`triggers` uses `|` between tokens, as returned by the API `trainedWords` field).
- **`top_triggers_by_frequency.txt`** — One trigger per line, **most common first** in this CSV (for `frequency_*` trigger banks). Regenerated by `curate_civitai_triggers.py`.
- **`model_names.txt`** — One **model title per line** (same row order as the CSV). For reference / search; not auto-injected into prompts.
- **`triggers_unique.txt`** — Optional: sorted unique triggers (generate locally; may be huge).

**Note:** The website filter `category=concept` is **not** available on the public models list API; this bank is the closest practical match (NSFW query + Illustrious/NoobAI bases).

## Regenerate / refresh

**Single search** (one `query=`, e.g. only `nsfw`):

```bash
python scripts/tools/fetch_civitai_nsfw_concepts.py --out data/civitai/nsfw_illustrious_noobai_models.csv --max-batches 40 --sleep 0.25
```

**Extended merge** — many site-style searches (see [`SEARCHES.md`](SEARCHES.md)), merged + deduped by model id, still filtered to **Illustrious / NoobAI**:

```bash
python scripts/tools/fetch_civitai_nsfw_concepts.py --preset extended --max-batches-per-query 8 --sleep 0.2 --out data/civitai/nsfw_illustrious_noobai_models.csv
```

Add more terms: `--extra-queries "foo,bar"`.

Then rebuild frequency list + embedded hot tags + optional name list:

```bash
python scripts/tools/curate_civitai_triggers.py --names-out data/civitai/model_names.txt
```

This updates:

- `data/civitai/top_triggers_by_frequency.txt`
- `utils/prompt/civitai_vocab.py` (`CIVITAI_HOT_TAGS`, top 150 triggers)

Optional API token (if you hit limits on fetch): `--token YOUR_KEY`.

## Use in prompts (`content_controls`)

With `safety_mode="nsfw"`:

- **`nsfw_civitai_pack`**
  - **`hits`** — All **`CIVITAI_HOT_TAGS`** (from the CSV, frequency-curated).
  - **`hits_lite`** — First 48 of that list (lighter prompt).
  - **`snippets`** / **`snippets_lite`** — Short fragments from CSV triggers + shortened **name** phrases (not full titles), deduped vs `hits`. Defined in `content_controls._CIVITAI_SNIPPET_TAGS` (refresh via `scripts/tools/extract_civitai_snippets_for_content_controls.py`).
  - Plus existing packs: `action`, `complex`, `clothing`, `objects`, `style`, etc.
- **`civitai_trigger_bank`**
  - **`light` / `medium` / `heavy`** — First N unique triggers scanning the **CSV in row order** (API order).
  - **`frequency_light` / `frequency_medium` / `frequency_heavy`** — First N lines from **`top_triggers_by_frequency.txt`** (popularity prior). Override path with **`civitai_frequency_txt`**.

CLI (`sample.py`): `--nsfw-civitai-pack`, `--civitai-trigger-bank`, `--civitai-model-bank-csv`, `--civitai-frequency-txt`.

## Plain-text trigger list (optional)

Write sorted unique triggers to `triggers_unique.txt` (small helper you can paste into a file and run):

```python
import csv
from pathlib import Path

src = Path("data/civitai/nsfw_illustrious_noobai_models.csv")
seen: set[str] = set()
uniq: list[str] = []
for row in csv.DictReader(src.open(encoding="utf-8")):
    for t in (row.get("triggers") or "").split("|"):
        x = t.strip()
        if x and x not in seen:
            seen.add(x)
            uniq.append(x)
Path("data/civitai/triggers_unique.txt").write_text("\n".join(sorted(uniq)), encoding="utf-8")
```
